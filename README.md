# ⚙️ Skript Systems

A collection of larger **Minecraft server systems** made with Skript.
Focused on clean gameplay, useful mechanics, and real server use.

---

## ✨ Included Systems

### ⚔️ FFA System

A lightweight free-for-all system for PvP servers.

**Features:**

* Join and leave messages
* Spawn-safe region protection
* Block placement tracking
* Protection against breaking map blocks
* Short and clean death messages
* Kill token reward system
* Admin command to clear placed block logs

**Command:**

* `/resetffalog` — Clears the placed block log

**Permission:**

* `ffa.admin`

---

### 💸 Money System

A banknote-based money system built around Essentials balance support.

**Features:**

* Withdraw money into physical banknotes
* Redeem banknotes by right clicking
* Multiple preset note values
* Balance checks and inventory checks
* Success and error sounds
* Kill reward money system

**Commands:**

* `/withdrawal <amount>`
* `/withdraw <amount>`
* `/banknote <amount>`

**Supported Values:**

* `10`
* `100`
* `500`
* `1000`
* `10000`

**Dependencies:**

* Essentials
* Vault

## ⚙️ Installation

1. Install **Skript** on your server
2. Install any required dependencies
3. Place the `.sk` files into:
   `/plugins/Skript/scripts/`
4. Run:

```text
/sk reload all
```
